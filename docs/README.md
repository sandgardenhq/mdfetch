# mdfetch API Documentation

This directory contains the auto-generated TypeDoc API documentation for mdfetch.

## Viewing Documentation

### Locally
Open `index.html` in your browser to view the documentation locally.

### GitHub Pages
The documentation is published via GitHub Pages at:
https://anthropics.github.io/mdfetch/

## Regenerating Documentation

To regenerate the documentation after making changes to the code:

```bash
npm run docs
```

This will update all files in this directory with the latest API documentation.

## Configuration

Documentation is configured via `typedoc.json` in the project root.

The `.nojekyll` file ensures GitHub Pages serves the documentation without Jekyll processing.
